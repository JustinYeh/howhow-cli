## Usage

在 iterm 上安裝
```shell
$ npm i howhow -g
```

相關操作可以在後頭加上 `-h` 參數
```shell
howhow -h
```

指示就會跳出～
```shell
  howhow command line tool

  Examples
  $ howhow
  喔喔喔喔喔喔喔原來是旺梨小鎮的土鳳梨酥啊！

  Show version
  $ howhow -v
  version is 1.0.64

  Open howhow youtube channel with Google Chrome
  $ howhow open
```
